🚀 الخطوة 1: حل مشكلة 404 (GitHub Pages)
═══════════════════════════════════════════════════════════════════

⚠️ إذا كنت تواجه خطأ 404، السبب هو:
المشكلة: الملفات موجودة في مجلد "website" بدلاً من الجذر!

الحل السريع:
━━━━━━━━━━━━

1️⃣ افتح: https://github.com/kkraariq-oss/Lavish-Center

2️⃣ تأكد من البنية:

   ✅ الصحيح:
   Lavish-Center/
   ├── index.html      ← في الجذر مباشرة
   ├── style.css
   └── ...

   ❌ الخطأ:
   Lavish-Center/
   └── website/        ← مجلد فرعي!
       ├── index.html
       └── ...

3️⃣ نقل الملفات للجذر:

   **عبر Git (الأسهل):**
   ```bash
   git clone https://github.com/kkraariq-oss/Lavish-Center.git
   cd Lavish-Center
   mv website/* .
   mv website/.nojekyll .
   rm -rf website
   git add .
   git commit -m "Fix: Move files to root"
   git push origin main
   ```

   **عبر GitHub Web:**
   - ادخل مجلد website
   - حمّل كل ملف
   - ارفعه في الجذر (خارج website)
   - احذف مجلد website

4️⃣ تفعيل GitHub Pages:
   - Settings → Pages
   - Source: main
   - Folder: / (root)
   - Save
   - انتظر 2-5 دقائق

5️⃣ اختبر:
   افتح: https://kkraariq-oss.github.io/Lavish-Center/
   
   ✅ يجب أن يظهر الموقع الآن!

🎨 الخطوة 2: إنشاء الأيقونات (إلزامي!)
═══════════════════════════════════════════════════════════════════

بدون الأيقونات، التطبيق لن يعمل بشكل صحيح على iPhone!

الأيقونات المطلوبة (الحد الأدنى):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. apple-touch-icon.png (180×180)     ← الأهم! ⭐
2. icon-512.png (512×512)
3. icon-192.png (192×192)

الأيقونات الكاملة (موصى بها):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
راجع ملف: ICON_SIZES_GUIDE.md
للحصول على قائمة كاملة بـ 19 قياس

طرق إنشاء الأيقونات:
━━━━━━━━━━━━━━━━━━━━━━

🌟 **الطريقة 1: استخدام Python Script (الأسرع)**
   ```bash
   pip install Pillow
   python3 create_all_icons.py
   ```
   ✅ سيتم إنشاء جميع الأيقونات تلقائياً!

🌟 **الطريقة 2: Favicon Generator (الأسهل)**
   1. اذهب إلى: https://realfavicongenerator.net/
   2. ارفع icon.svg أو صورة 512×512
   3. اختر الإعدادات:
      • iOS: تصميم مسطح بدون حواف
      • Android: اختر Maskable
      • Background: #ff6b9d
   4. اضغط "Generate"
   5. حمّل جميع الأيقونات
   6. ارفعها إلى GitHub

🌟 **الطريقة 3: Canva (تصميم مخصص)**
   1. افتح: https://www.canva.com/create/logos/
   2. أنشئ تصميم 512×512 بكسل
   3. استخدم الألوان:
      • Gradient: #ff6b9d → #c44569
      • نص/أيقونة: أبيض أو ذهبي
   4. أضف حرف "L" بخط عريض
   5. أضف أيقونة جوهرة 💎
   6. حمّل PNG بجودة عالية
   7. استخدم أداة تصغير لإنشاء الأحجام الأخرى:
      https://www.iloveimg.com/resize-image

🌟 **الطريقة 4: ImageMagick (لينكس/ماك)**
   ```bash
   # تثبيت ImageMagick
   sudo apt-get install imagemagick
   
   # تحويل SVG لجميع الأحجام
   convert icon.svg -resize 180x180 apple-touch-icon.png
   convert icon.svg -resize 512x512 icon-512.png
   convert icon.svg -resize 192x192 icon-192.png
   # ... إلخ
   ```

🔧 حل المشاكل الشائعة
═══════════════════════════════════════════════════════════════════

❌ المشكلة 1: تظهر أيقونة عامة (لقطة شاشة)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✅ **السبب:** apple-touch-icon.png غير موجودة
✅ **الحل:**
   - تأكد من رفع apple-touch-icon.png (180×180)
   - امسح التطبيق من الشاشة الرئيسية
   - أعد تثبيته من جديد

❌ المشكلة 2: التطبيق يفتح في Safari
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✅ **السبب:** manifest.json غير صحيح
✅ **الحل:**
   - تأكد من رفع manifest.json المحدث
   - تأكد من وجود "display": "standalone"
   - امسح Cache Safari

❌ المشكلة 3: لا يعمل بدون إنترنت
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✅ **السبب:** Service Worker غير نشط
✅ **الحل:**
   - افتح التطبيق مرة واحدة مع إنترنت
   - انتظر حتى يتم التحميل الكامل
   - الآن سيعمل بدون إنترنت

❌ المشكلة 4: الموقع لا يفتح (404)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✅ **السبب:** GitHub Pages غير مفعل أو إعدادات خاطئة
✅ **الحل:**
   - راجع "الخطوة 1" أعلاه
   - راجع ملف: GITHUB_PAGES_FIX.md

❌ المشكلة 5: لا يظهر زر "إضافة إلى الشاشة الرئيسية"
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✅ **السبب:** تستخدم Chrome بدلاً من Safari
✅ **الحل:**
   - استخدم Safari (المتصفح الأزرق الافتراضي)
   - في Safari: زر المشاركة  → إضافة إلى الشاشة الرئيسية

❌ المشكلة 6: الصفحة تظهر لكن بدون تنسيق
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✅ **السبب:** ملف style.css لم يتم تحميله
✅ **الحل:**
   - تأكد من رفع style.css في نفس مجلد index.html
   - امسح Cache: Settings → Safari → Clear History and Website Data
